Welcome to my dotfiles (?!)

After a lot of distrohopping, here are some ways to simplify reproducing a basic working [Linux] system

1. Remap ~Caps Lock~ to ~Ctrl~
2. No separate home folder but they are symlinked to a different partition, shared by multiple distros
   #+begin_src sh
ln -s /mnt/shared/Documents Documents
ln -s /mnt/shared/Downloads Downloads
ln -s /mnt/shared/dotfiles/.emacs.d .emacs.d
ln -s /mnt/shared/appimages appimages
ln -s /mnt/shared/Zotero Zotero
ln -s /mnt/shared/Pictures Pictures          
   #+end_src

4. Build emacs
   - install build dependencies
   - download tar from archive
   - build emacs
      #+begin_src sh
./autogen.sh 
./configure --with-native-compilation CFLAGS='-O3 -march=native'
 make clean
 make -j8
 make install
 #+end_src
